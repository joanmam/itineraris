📲 Instal·la l'app al mòbil

Pots afegir aquesta pàgina a la pantalla d'inici per usar-la com una app.

  • iPhone/iPad: Safari → botó Compartir → "Afegir a pantalla d'inici"
  • Android: Chrome → menú ⋮ → "Afegir a pantalla d'inici"
📤 Exportar dades
Guarda totes les rutes, parades, fotos i coordenades en un fitxer JSON. Pots guardar-lo a Drive o enviar-lo a un altre dispositiu.
—
📥 Importar dades
Carrega un fitxer exportat des d'un altre dispositiu. Les rutes noves s'afegiran sense esborrar les actuals.
—
ℹ️ Com sincronitzar entre dispositius
1. Exporta des del dispositiu A → guarda el JSON a Drive o envia'l per WhatsApp / Mail.
2. Obre el JSON al dispositiu B → importa'l amb el botó de dalt.
3. Les rutes es fusionen automàticament sense duplicats.
⚠️ Zona perillosa
Esborra totes les dades d'aquest dispositiu. Irreversible.
